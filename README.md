# vmware-datastores

Python Web application based on the Flask microframework to monitor disk usage
on VMware datastores using the pyVmomi SDK.

## Requirements

* Python >= 2.7.13
* Flask >= 1.0.2
* PostgreSQL >= 10.3
* pyVmomi >= 6.7.0
* An LDAP server for authentication
* An HTTP server as a front-end
